# Spitter Infected

This infected is a type of 'special' infected. A special infected has special abilities that a
common infected does not have. They are harder to kill, and usually do more damange to the player.


## Enemy Card

```
{
    name: "spitter",
    health: 20,
    damage: 50,
    velocity: 150,
}
```

## Movement Details

This enemy stands still

## Special ability  

This infected fires a projectile at the player at a random frequency. The projectile will travel in
a straight line once fired at the player. 

```
{
    projectile_velocity: 300,
}
```
