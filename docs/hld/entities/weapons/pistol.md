# Pistol

## Weapon Card

```
{
    mag_size: 15,
    reload_time: 2000,
    fire_rate: 10,
    velocity: 500,
    damage: 2,
}
```

### Details

The pistol fires a single bullet on each click. The bullets fly in a straight line based on where
the user was aiming at the time of the click. Bullets should _always_ appear as being shot by the
gun rather than spawning at the player. 

### Sprite / Design

At the moment, for testing, we will use `docs/art/weapons/sniper` as the design for the pistol.
