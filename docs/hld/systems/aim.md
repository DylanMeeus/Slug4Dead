## Aim (aiming in the game)

Aiming in the game is done by moving the mouse. The player can shoot in a 360 degree circle around
the player. While actively playing in a level (i.e, a level is loaded and the player state is alive)
we will render a crosshair where the player is aiming.

The crosshair effectively replaces the mouse, and thus moves along with the mouse. 

When the user clicks, a bullet is fired. The bullets travel as defined for each weapon.
